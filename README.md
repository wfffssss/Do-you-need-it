# do_you_need_it

Одностраничный сайт аукциона необычных вещей **«Оно тебе надо»**.

Проект выполнен на **HTML** и **CSS** без использования фреймворков и библиотек.

## Запуск

Откройте файл `index.html` в браузере или используйте расширение **Live Server** в VS Code.

## Структура проекта

```text
.
├── index.html
├── styles/
├── fonts/
├── images/
└── README.md
```

## Используемые технологии

* HTML5
* CSS3
* Локальные шрифты

## Особенности

* Семантическая верстка
* Адаптация под ширину от 1100px
* Локальное хранение шрифтов и изображений
